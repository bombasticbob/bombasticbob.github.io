Detailed Description

Utility functions and structures associated with file I/O, buffering, file info, and directory searching

These functions are designed specifically to provide a cross-platform capable method of dealing with some specific I/O and file-related tasks, such as directory searches, file information, buffered and non-buffered I/O, and canonical paths.

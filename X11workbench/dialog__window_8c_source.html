417 // No Input False Absent

421 //
422 // No Input - The application never expects to receive focus.
423 //
424 // Passive - The application will get focus but it will not grab it on

432 //

434 // window manager. Instead it will determine when it wants focus (from
435 // WM_TAKE_FOCUS suggestions given by the window manager, or from events
436 // such as ButtonPress or KeyPress). The application will then acquire

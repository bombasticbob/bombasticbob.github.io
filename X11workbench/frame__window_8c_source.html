515 // No Input False Absent
516 // Passive True Absent
517 // Locally Active True Present
518 // Globally Active False Present
519 //
520 // No Input - The application never expects to receive focus.
521 //
522 // Passive - The application will get focus but it will not grab it on
523 // its own. Instead it will only get focus if the window manager gives it
524 // to it. The application cannot help determine if it wants the focus or
525 // not at the given time/situation.

527 // Locally Active - The same as passive, but the application will also
528 // give focus to other windows that it owns of its own free will, without
529 // any interaction with the window manager, using XSetInputFocus.
530 //
531 // Globally Active - The application will not receive focus from the
532 // window manager. Instead it will determine when it wants focus (from
533 // WM_TAKE_FOCUS suggestions given by the window manager, or from events
534 // such as ButtonPress or KeyPress). The application will then acquire
535 // focus on its own using XSetInputFocus.

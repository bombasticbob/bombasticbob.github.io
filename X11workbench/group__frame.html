Detailed Description

The 'Frame Window' API for the X11workbench Toolkit identifies those functions that implement 'Frame Windows'. A Frame Window is a top-level window that is registered with the window manager and typically has a title bar, sizeable border, menu, and icon. It can contain other windows and manage menu events.
A special case of a frame window is a dialog frame window. See DLGCreateDialogWindow()
